DevOps software package which can develop, secure, and operate software.
Alternative to [[github]].

Includes:

- Git repository
- CI/CD
- Issue tracking
- Code review (merge requests)
- Project management
- Wiki
- Pages for static sites / documentation
- Container registry
- Can be [[self-hosting (web services)|self-hosted]]
