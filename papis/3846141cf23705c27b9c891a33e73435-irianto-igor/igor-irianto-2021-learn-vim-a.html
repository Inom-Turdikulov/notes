Learn Vim the Smart Way

What's This?

Learn Vim (the Smart Way) is a guide to learn the good parts of Vim. This project was inspired by Steve Losh' Learn Vimscript the Hard Way. I thought it would be neat to do a broader, albeit less deep, overview of Vim + Vimscript in 2023. So here it is!

This guide consists of three parts:

  • How to use Vim
  • How to customize Vim
  • How to use Vimscript

If you're an advanced user already, I would encourage you to read this guide from start to finish anyway, because you will learn something new. Let's get started!
